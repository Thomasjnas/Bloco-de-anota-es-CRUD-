## 📌 Sobre o Projeto

Este projeto consiste em uma aplicação Full Stack de gerenciamento de tarefas desenvolvida com:

- Front-end: Vue.js + Tailwind CSS
- Back-end: ASP.NET Core Web API
- Comunicação: REST (JSON / HTTP)

A aplicação implementa um CRUD completo, permitindo ao usuário:

- Criar tarefas
- Visualizar tarefas
- Editar tarefas
- Marcar como concluídas
- Excluir tarefas

O objetivo do projeto é demonstrar domínio de integração entre front-end e back-end, organização de código, arquitetura web e manipulação de estado no cliente.
